# 🚀 Apex Digital - Modern Business Website

यह Python और Streamlit का उपयोग करके बनाई गई एक आधुनिक, रेस्पॉन्सिव और क्लाइंट-रेडी बिजनेस वेबसाइट है।

## ✨ फीचर्स (Features)
- 📱 **Fully Responsive:** मोबाइल और डेस्कटॉप दोनों पर परफेक्ट दिखती है।
- 🎯 **Multi-page Navigation:** होम, सर्विसेज और कांटेक्ट पेज।
- 📊 **Lead Generation Form:** ग्राहकों से संपर्क करने के लिए एक्टिव फॉर्म।

## 🛠️ टेक स्टैक (Tech Stack)
- **Language:** Python
- **Framework:** Streamlit
